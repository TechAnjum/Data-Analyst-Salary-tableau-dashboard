# Data Analyst Salary Analysis Dashboard

This project presents an interactive Tableau dashboard analyzing global Data Analyst salary trends based on experience level, employment type, company size and geographic location.

## Tools Used

- Tableau
- Data Visualization
- Data Analysis

## Dataset

The dataset contains information about salaries in data-related roles across multiple countries including:

- Job Title
- Experience Level
- Employment Type
- Company Location
- Salary in USD
- Company Size

## Key Insights

- Senior professionals receive significantly higher salaries compared to entry-level roles.
- The United States has the highest number of data professionals.
- Medium sized companies dominate hiring trends.
- Salary varies significantly based on experience level and company location.

## Dashboard Features

- Average salary by experience level and employment type
- Global map showing salary distribution by country
- Company size distribution
- Experience level analysis
- Top employee residence countries
- Salary comparison across job roles
